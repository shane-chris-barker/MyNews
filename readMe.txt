Plugin searches the Bing Api for a given news category in a given language and displays the results in the admin area of Wordpress

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings- My News screen to configure the plugin
4. Enter a valid Bing News Api Key and view the latest news from the admin area of your Wordpres site.
