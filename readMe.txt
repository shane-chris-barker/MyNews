Plugin searches the Bing Api for a given news category in a given language and displays the results in the admin area of Wordpress.

PLEASE NOTE:

When used, this plugin sends the selected news query to the Microsoft Cognitive Services API, where the search will be processed and results returned to the plugin.

The Cognitive API is part of Microsoft's Azure service. A service offering cloud based services.
For more information on the Microsoft Azure service, please visit https://azure.microsoft.com/

For more information on the Microsoft Cognitive Services API that this plugin uses, please visit https://azure.microsoft.com/services/cognitive-services/

More specifically, the plugin makes use of the Bing Search Service API aspect of the Cognitive Services feature. You can find information on the Bing services
API by visiting https://azure.microsoft.com/services/cognitive-services/bing-web-search-api/


For Privacy and data protection information related to the Cognitive API and Microsoft, please visit https://www.microsoft.com/TrustCenter/Privacy/default.aspx

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/my_news` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the plugins -> My News screen to configure the plugin.
4. Enter a valid Bing News Api Key and view the latest news from the admin area of your Wordpress site.
